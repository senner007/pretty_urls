# pretty_urls

Single-page app and pretty urls with MVC, js(or jQuery), bootstrap and php.
[Demo](http://www.codetoshow.com)

TO RUN:
parcel index.html

TO BUILD:
parcel build src/main.js
insert php code from index.php to dist/index.html and rename to index.php
copy dist to ftp
