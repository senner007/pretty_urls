# pretty_urls

Single-page app and pretty urls with MVC, js(or jQuery), bootstrap and php.
[Demo](http://www.codetoshow.com)

#### TO RUN:
parcel index.html

#### TO BUILD:
- parcel build src/main.js
- rename /dist/index.html to /dist/index.php
- copy /dist to ftp
